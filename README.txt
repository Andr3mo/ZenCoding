= SETTINGS =

    == zen-settings.sublime-settings ==

        Look in $PACKAGES_PATH/ZenCoding/zen-settings.sublime-settings for
        keys and their default values.

        Create a $PACKAGES_PATH/User/zen-settings.sublime-settings to place any
        overrides in.

        === my_zen_settings ===

        See https://github.com/sergeche/zen-coding/blob/master/python/zencoding/zen_settings.py
        to see what settings you can override/extend.
